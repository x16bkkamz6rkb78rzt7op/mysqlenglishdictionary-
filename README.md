mysqlenglishdictionary-
=======================

A  dictionary with 176023 entries. Text was extracted from the files at http://www.mso.anu.edu.au/~ralph/OPTED/ and then parsed and stored in a 16MB MySQL database. The database has three fields : a. word b. wordtype and c. definition. You can use this standalone or as a jquery/ajax/PHP/etc addon for your programs.
